SQL Comment Generator

Convierte enunciados de ejercicios en comentarios SQL multilínea listos para usar

¿Qué hace esta herramienta?
1
Pega tus enunciados Escribe o pega todos los enunciados de tu ejercicio SQL en el área de texto, uno por línea.
2
Genera los comentarios El algoritmo envuelve cada línea automáticamente entre /* y */, separados por una línea en blanco para mayor legibilidad.
3
Copia y pega en tu editor Usa el botón Copiar para llevar el resultado directo a tu editor SQL. Cada enunciado queda listo para escribir la consulta justo debajo.
4
Guarda en el historial Pulsa Guardar en Historial para conservar el resultado. El historial persiste aunque cierres el navegador (usa localStorage).
Ejemplo de entrada
Calcula la suma del presupuesto de todos los departamentos.
Calcula la media del presupuesto de todos los departamentos.
→
/*
Calcula la suma del presupuesto de todos los departamentos.
*/

/*
Calcula la media del presupuesto de todos los departamentos.
*/
